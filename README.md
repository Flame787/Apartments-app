# Description:

Full‑stack application built with React and Express for browsing and renting holiday accommodations

Tech stack:

Frontend: React (TypeScript) + React Router, Redux Toolkit, TanStack Query, SCSS

Backend: Node.js, Express.js


# Installation:

git clone https://github.com/Flame787/Apartments-app.git

- Open the folder where you installed the code.  

- Install dependencies in both 'frontend' and 'backend' folder:

cd frontend
npm install

(cd ..)

cd backend
npm install


# Run the app locally:

- Navigate to frontend in one terminal:

cd frontend
npm run dev

- Navigate to backend in another terminal:

cd backend
npm run dev
